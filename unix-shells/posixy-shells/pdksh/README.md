# modulinos/pdksh

A modulino in `pdksh`

# EXAMPLE

```console
$ ./lib/modulino
Main: The meaning of life is 42

$ ./lib/test
Test: The meaning of life is 42
```

# REQUIREMENTS

* [pdksh](https://directory.fsf.org/wiki/Pdksh)

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
