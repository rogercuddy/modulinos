# modulinos/dash

A modulino in `dash`

# EXAMPLE

```console
$ ./lib/modulino
Main: The meaning of life is 42

$ ./lib/test
Test: The meaning of life is 42
```

# REQUIREMENTS

* [dash](http://www.in-ulm.de/~mascheck/various/ash/)

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
