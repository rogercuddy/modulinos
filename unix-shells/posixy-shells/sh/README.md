# modulinos/sh

A modulino in POSIX `sh`

# EXAMPLE

```console
$ ./lib/modulino
Main: The meaning of life is 42

$ ./lib/test
Test: The meaning of life is 42
```

# REQUIREMENTS

* [coreutils](https://www.gnu.org/software/coreutils/coreutils.html)

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
