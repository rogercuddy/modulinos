# modulinos/ksh88

A modulino in `ksh88`

# EXAMPLE

```console
$ ./lib/modulino
Main: The meaning of life is 42

$ ./lib/test
Test: The meaning of life is 42
```

# REQUIREMENTS

* [ksh88](http://www.kornshell.com/), sometimes packaged as "ksh".

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
