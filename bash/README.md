# modulinos/bash

A modulino in `bash`

# EXAMPLE

```
$ bash lib/modulino
Main: The meaning of life is 42

$ bash lib/test
Test: The meaning of life is 42
```

# REQUIREMENTS

* [bash](https://www.gnu.org/software/bash/)
