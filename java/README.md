# modulinos/java

Modulino in Java

# EXAMPLE

```
$ mvn package
$ bin/modulino
Main: The meaning of life is 42
$ bin/test
Test: The meaning of life is 42
```

# REQUIREMENTS

* [JDK](http://www.oracle.com/technetwork/java/javase/downloads/index.html) 1.7+

# CODE COVERAGE

```
$ mvn site
$ open target/site/coburtura/index.html
```
